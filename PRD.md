# PRD - MACIF Blue Team

## Vision
Fournir un kit Blue Team simple pour detection et reponse basiques.

## Probleme
Les equipes ont besoin de scripts concrets pour demos et formation.

## Utilisateurs
- Blue Team / SOC
- Etudiants cyber

## MVP (fonctionnalites)
- Anonymisation GDPR
- Audit comptes inactifs (PowerShell)
- Score phishing (mots-cles)
- File Integrity Monitor
- Generateur MDP conforme

## Evolutions
- Playbooks d'incident
- Rapports PDF/JSON
- Dashboard simple

## KPI
- Nombre de scenarii couverts
- Temps d'execution

## Entrees / Sorties
- Entrees: CSV/JSON sample
- Sorties: CSV/JSON + logs

## Contraintes
- Scripts multi-OS (Python/PowerShell)

## Hors perimetre
- Detection avancee EDR
